### Step 1: List Every Recurring Task

- Competitive research
- Invoice/payment follow-up

### Step 2: Score Each Task

Rate each task on two dimensions:

**Repeatability (1-5):** How consistent and predictable is this task?
- 1 = Every instance is unique, requires deep judgment
- 3 = Has a pattern but needs some customization each time
- 5 = Identical every time, pure process

**AI Capability (1-5):** How well can current AI handle this?
- 1 = AI can't do this (physical work, high-stakes negotiation)
- 3 = AI can do a draft that needs human review
- 5 = AI can handle end-to-end with minimal oversight

| Task | Repeatability | AI Capability | Score (R × A) | Priority |
|------|--------------|---------------|---------------|----------|
| Answer customer emails | 4 | 4 | 16 | 🟢 High |
| Write social posts | 4 | 4 | 16 | 🟢 High |
| Research competitors | 3 | 5 | 15 | 🟢 High |
| Write proposals | 2 | 3 | 6 | 🔴 Low |
| Invoice clients | 5 | 2 | 10 | 🟡 Medium |

**Scores above 12:** Automate first
**Scores 8-12:** Automate second wave
**Scores below 8:** Keep human or partially automate

## Part 4: The 90-Day Implementation Plan

### Week 1-2: Quick Wins
Pick your **top 3 tasks** (highest automation score) and implement using the simplest tool:
- [ ] Set up ChatGPT/Claude with a custom prompt for Task 1
- [ ] Create a Zapier automation for Task 2
- [ ] Build a template/prompt library for Task 3
- [ ] Measure time savings (track actual hours before/after)

### Week 3-4: Build Habits
- [ ] Make AI tools part of your daily workflow (not a separate step)
- [ ] Train team members on the tools (screen recording of your process)
- [ ] Document your prompts that work well (prompt library)
- [ ] Review: are you actually saving time? Adjust if not.

### Month 2: Second Wave
- [ ] Automate your next 3-5 tasks
- [ ] Connect tools together (Zapier/Make workflows)
- [ ] Build more sophisticated prompts with context
- [ ] Consider upgrading to more powerful tools if ROI justifies it

### Month 3: Systematize
- [ ] Document all AI workflows in one place (SOP document)
- [ ] Set up monitoring (are automations still working?)
- [ ] Calculate actual ROI vs. projections
- [ ] Identify next automation opportunities
- [ ] Consider: do you need a multi-agent system? (If so, talk to us)

## Part 5: Prompt Templates by Function

### Email Response
```
Context: I run a [business type]. A customer emailed about [topic].

Customer email:
[paste email]

Write a professional response that:
1. Acknowledges their concern
2. Provides a clear answer or next step
3. Maintains a [warm/professional/casual] tone
4. Keeps it under 150 words
```

### Social Media Post
```
Create a [platform] post about [topic] for my [business type].

Audience: [describe your audience]
Goal: [awareness/engagement/conversion]
Tone: [professional/casual/inspirational/educational]
Include: [hashtags/CTA/emoji preference]
Length: [platform-appropriate]

Do not use: [words/phrases to avoid]
```

### Meeting Summary
```
Summarize this meeting transcript into:
1. Key decisions made (bullet points)
2. Action items with owners and deadlines
3. Open questions that need follow-up
4. Next meeting topics

Transcript:
[paste transcript]
```

### Competitive Research
```
Research [competitor name] and provide:
1. Their main products/services and pricing
2. Recent news or announcements (last 90 days)
3. Their strengths vs. our offering
4. Their weaknesses or gaps we can exploit
5. Key takeaway for our strategy

Our business: [brief description]
Our differentiator: [what makes you different]
```

### Content Repurposing
```
I have this [blog post/podcast transcript/video script]:

[paste content]

Repurpose this into:
1. Three tweet-length posts (max 280 chars each)
2. One LinkedIn post (150-300 words)
3. One email newsletter paragraph (100 words)
4. Five Instagram story talking points

Maintain my voice: [describe your voice]
```

## Part 6: Decision Trees

### "Should I Automate This?"
```
Is this task recurring? 
  → No → Don't automate (one-off tasks aren't worth the setup)
  → Yes ↓

Does it follow a consistent pattern?
  → No → Partially automate (AI drafts, human finishes)
  → Yes ↓

Does it require real-time human judgment?
  → Yes → AI assists, human decides
  → No ↓

Is the cost of error low?
  → No → AI drafts, human reviews before sending
  → Yes → Fully automate with monitoring
```

### "Which Tool Should I Use?"
```
Do you need to connect two existing tools?
  → Yes → Zapier or Make
  → No ↓

Do you need to generate content?
  → Yes → ChatGPT, Claude, or Jasper
  → No ↓

Do you need to analyze/summarize documents?
  → Yes → Claude (best for long documents)
  → No ↓

Do you need multiple AI agents working together?
  → Yes → OpenClaw, LangChain, or CrewAI
  → No → Start with ChatGPT Plus
```
