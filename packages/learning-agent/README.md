# Learning Agent

**Version:** 1.0 | **Setup Time:** 10 minutes

A structured learning partner that decompose any topic into a learnable curriculum, creates a realistic learning path, and schedules spaced repetition reviews — so you actually retain what you learn.

## Quick Start

1. Copy ROLE.md to your agents directory
2. Pick a topic you want to learn
3. Tell your agent: "Create a learning plan for [topic] — I want to [goal] within [timeframe]"
4. Follow the plan; let the agent schedule reviews

## What This Agent Does

- Decomposes complex topics into learnable chunks
- Creates personalized learning paths with realistic timelines
- Designs practice exercises (not just reading)
- Schedules spaced repetition for retention

## Files Included

| File | Purpose |
|------|---------|
| ROLE.md | Agent role and learning methodology |
| templates/topic-decomposition.md | Break any topic into learnable units |
| templates/learning-path.md | Full curriculum with timeline |
| templates/practice-exercises.md | Active learning exercise generator |
| templates/spaced-repetition.md | Review scheduling template |
