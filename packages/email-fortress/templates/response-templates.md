# Email Response Templates

10 common patterns. Fill in the brackets. Agent uses these for drafting.

---

## 1. Meeting Request — Accept

```
Hi [Name],

Works for me. [Day/time] is confirmed.

[Location/video link if applicable]

See you then.

[Signature]
```

---

## 2. Meeting Request — Decline / Reschedule

```
Hi [Name],

I'm not available [their proposed time], but I'd like to connect.

Are any of these windows open for you?
- [Option 1]
- [Option 2]
- [Option 3]

Let me know what works.

[Signature]
```

---

## 3. Invoice / Payment Confirmation

```
Hi [Name],

Confirming receipt of your invoice #[number] for $[amount].

Payment will be processed by [date] via [method].

Let me know if you need anything else.

[Signature]
```

---

## 4. Project Status Update Request

```
Hi [Name],

Quick update on [project]:

Current status: [one sentence]
What's next: [one sentence]  
Expected completion: [date or "TBD — will update you by [date]"]

Happy to jump on a call if you'd like more detail.

[Signature]
```

---

## 5. Information Request — Can Answer

```
Hi [Name],

[Direct answer to their question — 1-3 sentences]

Let me know if you need more detail.

[Signature]
```

---

## 6. Information Request — Need Time / More Info

```
Hi [Name],

Good question. I need to [check on X / get back to you on Y].

I'll have an answer for you by [date/time].

[Signature]
```

---

## 7. Thank You / Acknowledgment

```
Hi [Name],

Thanks for [the introduction / the quick turnaround / sharing this].

[One sentence follow-up if needed.]

[Signature]
```

---

## 8. Follow-Up (No Response)

```
Hi [Name],

Following up on my note from [date]. Wanted to make sure this didn't get buried.

[1-line summary of original ask or offer]

Let me know when you have a moment.

[Signature]
```

---

## 9. Delivery / Completion Notice

```
Hi [Name],

[Deliverable] is complete and ready for your review.

[Link / attachment / location]

Next steps: [what they need to do, or what happens next]

Let me know if you have questions.

[Signature]
```

---

## 10. Hard No (Polite Decline)

```
Hi [Name],

Thanks for reaching out. After reviewing, [project/request/partnership] isn't a fit for where we're focused right now.

[Optional: specific reason, or "I want to be respectful of your time."]

Best of luck with it.

[Signature]
```
