# Competitive Intelligence Agent

**Version:** 1.0 | **Setup Time:** 15 minutes

Monitor your competitors automatically. Track their YouTube content, social posts, product changes, and pricing — delivered as a daily digest.

## Quick Start

1. Copy ROLE.md to your agents directory
2. Fill in competitor list in `scripts/youtube-competitor-monitor.md`
3. Set up the daily digest cron (see `crons/daily-competitive-digest.md`)
4. Test: "Give me a competitive brief on [Competitor Name]"

## Files Included

| File | Purpose |
|------|---------|
| ROLE.md | Agent role and research constraints |
| scripts/youtube-competitor-monitor.md | YouTube RSS tracking setup |
| scripts/social-scanning.md | Social media monitoring guide |
| crons/daily-competitive-digest.md | Automated daily brief |
| templates/competitor-report.md | Structured output template |
