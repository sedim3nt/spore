# Freelancer Ops Kit

**Version:** 1.0 | **Setup Time:** 20 minutes

Client communications, invoicing, proposals, and tax prep — handled by your agent. Stop losing deals to slow follow-ups and stop losing money to chaotic invoicing.

## Quick Start

1. Copy ROLE.md to your agents directory
2. Customize the rate calculator with your actual rates
3. Set up the tax-prep reminder cron
4. Test: "Draft a proposal for [client] for [project description]"

## Files Included

| File | Purpose |
|------|---------|
| ROLE.md | Agent role and client protocols |
| templates/invoice.md | Invoice template |
| templates/proposal-sow.md | Proposal + Statement of Work template |
| templates/client-comms.md | 8 common client communication patterns |
| templates/rate-calculator.md | Rate setting and project scoping guide |
| crons/tax-prep-reminder.md | Quarterly tax reminder cron |
