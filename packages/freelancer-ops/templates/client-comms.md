# Client Communication Templates

8 common patterns. Fill in the brackets.

---

## 1. Project Kick-Off Email

```
Subject: [Project Name] — Kick-off

Hi [Name],

Excited to get started. Here's what happens next:

1. I'll [first deliverable] by [date]
2. I'll need from you: [materials / access / decisions] by [date]
3. Our first check-in: [proposed date/time]

Best way to reach me: [email/channel]. I'll respond within [X] hours on business days.

Let me know if you have questions before we begin.

[Your name]
```

---

## 2. Status Update (Proactive)

```
Subject: [Project Name] — Update

Hi [Name],

Quick update on where things stand:

✅ Done: [what's complete]
🔄 In progress: [what you're working on now]
⏳ Next: [what's coming]

On track for [delivery date]. [Or: I'm running [X] days behind because [brief reason] — new target: [date].]

No action needed from you right now. I'll send [next deliverable] by [date].

[Your name]
```

---

## 3. Feedback Request

```
Subject: [Project Name] — [Deliverable] ready for review

Hi [Name],

[Deliverable] is ready: [link / attached]

Specifically looking for your feedback on:
- [Specific question 1]
- [Specific question 2]

Please send feedback by [date] to keep us on schedule.

[Your name]
```

---

## 4. Scope Creep Discussion

```
Subject: [Project Name] — Quick question on scope

Hi [Name],

As we've gotten into the project, [new request / expanded requirement] has come up.

That's outside our original scope (which covered [original scope summary]).

Two options:
1. Include it at $[X] additional — I can add it to invoice #[next]
2. Defer it to a Phase 2

Happy to discuss either. Which works better for you?

[Your name]
```

---

## 5. Delay Notification (Your Side)

```
Subject: [Project Name] — Brief delay

Hi [Name],

I wanted to give you a heads up: I'll need [X additional days] on [deliverable].

Reason: [brief, honest explanation]

New target: [date]. The rest of the timeline remains on track.

Sorry for any disruption. Let me know if this causes issues on your end.

[Your name]
```

---

## 6. Client Is Late (Gentle Nudge)

```
Subject: [Project Name] — Following up

Hi [Name],

Just following up on [the thing you were waiting on] — I need it to keep the project moving.

I've been holding [work] until I receive [what you need]. If I have it by [date], we can still hit [milestone].

Let me know if anything's changed on your end.

[Your name]
```

---

## 7. Payment Follow-Up

```
Subject: Invoice #[number] — Payment reminder

Hi [Name],

Just a reminder that Invoice #[number] for $[amount] was due [date].

[If first reminder:] Happy to help if there's any question about it.
[If second+ reminder:] Please let me know when to expect payment. I'm happy to work through any issues.

Payment methods: [methods]

[Your name]
```

---

## 8. Project Complete / Wrap-Up

```
Subject: [Project Name] — Complete ✓

Hi [Name],

[Project] is wrapped up. Final deliverables:

- [Deliverable 1]: [link]
- [Deliverable 2]: [link]

[If final invoice:] Final invoice #[number] for $[amount] is attached. Payment due [date].

It was great working with you on this. [One genuine sentence about the project or what you built together.]

If you need anything in the future, I'd love to work together again.

[Your name]
```
